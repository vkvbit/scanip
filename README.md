# scanip
A bash script to scan a IP address with different tools.

## How to use

### Step 1:
```
sudo apt-get install -y nikto
pip3 install aort
git clone https://github.com/vkvbit/scanip.git
cd scanip
```

### Step 2:

```
chmod a+x scanip.sh
```

### Step 3:
```
nano scanip.sh
```
Change value of target_ip enclosed in single quotes and save the file

### Step 4:
```
./scanip.sh
```
